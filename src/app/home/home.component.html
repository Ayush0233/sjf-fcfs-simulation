What is FCFS...?

First-Come First-Served (FCFS) is a CPU scheduling algorithm where the process that arrives first is selected to run first.

It is the simplest scheduling method, where processes are executed in the order they arrive without preemption. FCFS is easy to implement but can lead to longer average waiting times, especially when early processes have long burst times..

What is Shortest Job First...?

Shortest Job First (SJF) is a CPU scheduling algorithm where the process with the smallest execution time (burst time) is selected to run next.

It helps minimize the average waiting time for all processes and improves system efficiency. SJF can be either preemptive or non-preemptive.

Premptive Scheduling

  • CPU can be taken away any time.
  • More efficient
  • Faster
  • More complex

Non-Premptive Scheduling

  • Process runs till end
  • Less efficient
  • Slower
  • Simple
